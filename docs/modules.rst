#################
Modules
#################

In version 1.0, Harvard Mobile includes the following modules:

* People Directory
* Map
* Events (Gazette)
* Courses
* News (Gazette)
* Student Dining

In addition to the following utility modules:

* Schools (links to Harvard school web pages)
* Settings
* About
* Full Website (link to www.harvard.edu)
